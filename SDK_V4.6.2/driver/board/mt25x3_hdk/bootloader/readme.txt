Bootloader main module usage guide

Brief:          This module only for bootloader, it is bootloader implementation.
Usage:          GCC: Include the module with "include $(SOURCE_DIR)/driver/board/mt25x3_hdk/bootloader/module.mk" in your GCC bootloader project Makefile .
                KEIL: Drag the driver/board/mt25x3_hdk/bootloader/core/src folder to your bootloader project . Add driver/board/mt25x3_hdk/bootloader/core/inc to INCLUDE_PATH.
                IAR: Drag the driver/board/mt25x3_hdk/bootloader/core/src folder to your bootloader project . Add driver/board/mt25x3_hdk/bootloader/core/inc to INCLUDE_PATH.
Dependency:     N/A
Notice:         N/A

Project:        Please find the project under project folder with "bootloader".
