bt_android_notification
mt2523_hdk » apps
Overview
  • Application project description
    • This application demonstrates a simple and convenient way to access various types of notifications generated from Android notification center.
  • Features of the application
    • This application supports the notifications received in the notification center on the Android smartphone.
  • Process / procedure of the sample
    • Output from the application project: The notifications will show in the log.
Hardware and software environment
  • Supported platform
    • LinkIt 2523 HDK.
  • Remote device configuration
    • Install the MediaTek SmartDevice APK on an Android smartphone.
  • Environment configuration
    • The output logs are communicated through a micro USB cable to the PC from MK20 USB connector on the HDK.
    • Install the mbed serial driver according to the instructions at https://developer.mbed.org/handbook/Windows-serial-configuration. For more information, please refer to section "Installing the LinkIt 2523 HDK drivers on Microsoft Windows" on the "LinkIt 2523 HDK User Guide" in [sdk_root]/doc folder.
    • Use a type-A to micro-B USB cable to connect type-A USB of the PC and MK20 micro-B USB connector on the LinkIt 2523 HDK. For more information about the connector cable, please refer to https://en.wikipedia.org/wiki/USB#Mini_and_micro_connectors.
    • Launch a terminal emulator program, such as Tera terminal on your PC for data logging through UART. For the installation details, please refer to section "Installing Tera terminal on Microsoft Windows" on the "LinkIt for RTOS Get Started Guide" in [sdk_root]/doc folder.
    • COM port settings. baudrate: 115200, data bits: 8, stop bit: 1, parity: none and flow control: off.
Directory contents
  • Source and header files
    • src/main.c: Main program.
    • src/system_mt2523.c: MT2523x's system clock configuration file.
    • src/ept_gpio_var.c: GPIO pin assignment with variable names.
    • src/ept_eint_var.c: EINT assignment file with variable names.
    • src/bt_notify_test.c: App file, show notifications or send data to Andoird.
    • src/bt_init.c: This file includes bluetooth initialization example code.
    • src/sys_init.c: This file initializes the basic hardware environment for the system, including system clock, UART port for logging, Pinmux, cache and other necessary hardware.
    • src/hci_log.c: This file is for HCI log shown.
    • inc/task_def.h: Define the task stack size, queue length, project name, and priority for the application to create tasks.
    • inc/hal_feature_config.h: MT2523x's feature configuration file.
    • inc/memory_map.h: MT2523x's memory layout symbol file.
    • inc/ept_eint_drv.h: EINT basic configuration header file.
    • inc/ept_gpio_drv.h: GPIO pin assignment and basic configuration file.
    • inc/sys_init.h: The header file of sys_init.c.
    • inc/bt_init.h: The header file of bt_init.c.
    • inc/bt_log.h: The header file of bt logging APIs.
    • inc/FreeRTOSConfig.h: FreeRTOS feature configuration file.
    • inc/ept_keypad_drv.h: The KEYPAD configuration file generated by Easy Pinmux Tool(EPT). Please do not edit the file.
    • inc/cust_battery_meter.h: The customization file to configure the battery level parameters.
    • inc/cust_battery_meter_table.h: The customization file to configure the battery meter parameters.
    • inc/cust_battery_temperature_table.h: The customization file to configure the battery temperature parameters.
    • inc/cust_charging.h: The customization file to configure the charging parameters.
    • GCC/startup_mt2523.s: MT2523x's startup file for GCC.
    • GCC/syscalls.c: MT2523x's syscalls for GCC.
  • Project configuration files using GCC
    • GCC/feature.mk: Feature configuration.
    • GCC/Makefile: Makefile.
    • GCC/flash.ld: Linker script.
  • Project configuration files using Keil
    • MDK-ARM/bt_android_notification.uvprojx: uVision5 project file. Contains the project structure in XML format.
    • MDK-ARM/bt_android_notification.uvoptx: uVision5 project options. Contains the settings for the debugger, trace configuration, breakpoints, currently open files, etc.
    • MDK-ARM/flash.sct: Linker script.
    • MDK-ARM/startup_mt2523.s: MT2523x's startup file for Keil.
  • Project configuration files using IAR
    • EWARM/bt_android_notification.ewp: IAR project file. Contains the project structure in XML format
    • EWARM/bt_android_notification.eww: IAR workspace file. Contains project information.
    • EWARM/flash.icf: Linker script.
Run the demo
  • Build the application project with the command "./build.sh mt2523_hdk bt_android_notification" from the SDK root folder and download the binary file to the LinkIt 2523 HDK.
  • Connect your board to the PC with a micro USB-cable.
  • Power on the LinkIt 2523 HDK.
  • Connect to LinkIt 2523 HDK from Android smartphone using Mediatek SmartDevice APK
  • The notifications will be received.
  • The notification will show in the log on the connected SecureCRT or Hyper Terminal. Note: you could filter"Noti_S"&"Noti_E" to check the notification content.
