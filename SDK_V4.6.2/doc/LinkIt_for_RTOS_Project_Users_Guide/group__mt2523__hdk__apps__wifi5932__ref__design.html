wifi5932_ref_design
mt2523_hdk » apps
Overview
  • Example project description
    • This project demonstrates the Wi-Fi connectivity features of the LinkIt 5932 HDK through the following:
      • How to use the host device MT2523 to initialize the Wi-Fi.
  • Application features
    • Act as a Wi-Fi station to connect to a Wi-Fi network.
    • Ping out or into the device.
Hardware and software environment
  • Supported platform
    • MediaTek LinkIt 5932 HDK.
  • Pin connection between MT5932 HDK and MT2523 HDK +------------—+--------------------—+ | MT2523 HDK | MT5932 HDK | +------------—+--------------------—+ | J1001(2) | J2103(GPIO13) | +------------—+--------------------—+ | J1002(2) | J2103(GPIO14) | +------------—+--------------------—+ | J1003(2) | J2103(GPIO15) | +------------—+--------------------—+ | J1004(2) | J2103(GPIO16) | +------------—+--------------------—+ | J1005(2) | J2103(GPIO11) | +------------—+--------------------—+ | J1006(2) | J2103(GPIO12) | +------------—+--------------------—+ | Sensor(19) | J2101(GPIO4) | +------------—+--------------------—+ | Sensor(5) | GND | +------------—+--------------------—+ Table 1. Pin connection To run the project on the LinkIt 5932 HDK:
    • Connect MT5932 to MT2523 according to table 1.
    • In this mode, if the power is on, MT2523 will send the firmware to MT5932 by SDIO, then MT5932 will boot up.
  • There are three buttons on the board:
    • RST - reset.
    • EINT - external interrupt trigger.
    • RTC_INT - RTC interrupt trigger.

Environment configuration

  • A serial tool is required, such as hyper terminal or TeraTerm for UART logging.
  • COM port settings. baudrate: 115200, data bits: 8, stop bit: 1, parity: none and flow control: off.
Directory contents
  • Source and header files
    • src/main.c: Main program.
    • src/system_mt2523.c: The configuration file of the Cortex-M4 with floating point core registers and system clock.
    • src/sys_init.c: This file initializes the basic hardware environment for the system, including system clock, UART port for logging, Pinmux, cache and other necessary hardware.
    • src/ept_eint_var.c: The EINT configuration file generated by Easy Pinmux Tool (EPT). Please do not edit the file.
    • src/ept_gpio_var.c: The GPIO configuration file generated by the EPT. Please do not edit the file.
    • inc/at_command_wifi.c AT command for wifi operation
    • inc/atci_cmd.c AT command basic function
    • inc/lwip_network.c Network helper function for Wi-Fi station mode initialization until IP network ready.
    • inc/FreeRTOSConfig.h: FreeRTOS feature configuration file.
    • inc/task_def.h: Define the task stack size, queue length, project name, and priority for the application to create tasks.
    • inc/hal_feature_config.h: MT2523x feature configuration file.
    • inc/memory_map.h: MT2523x memory layout symbol file.
    • inc/ept_eint_drv.h: The EINT configuration file generated by the EPT. Please do not edit the file.
    • inc/ept_gpio_drv.h: The GPIO configuration file generated by the EPT. Please do not edit the file.
    • inc/ept_keypad_drv.h: The KEYPAD configuration file generated by the EPT. Please do not edit the file.
    • inc/cust_battery_meter.h: The customization file to configure the battery level parameters.
    • inc/cust_battery_meter_table.h: The customization file to configure the battery meter parameters.
    • inc/cust_battery_temperature_table.h: The customization file to configure the battery temperature parameters.
    • inc/cust_charging.h: The customization file to configure the charging parameters.
    • inc/sys_init.h: The header file of sys_init.c.
    • inc/at_command_wifi.h Prototype declaration for src/at_command_wifi.c.
    • inc/lwip_network.h Prototype declaration for src/lwip_network.c.
    • inc/lwipopts.h The configuration file of tcpip feature.
    • GCC/startup_mt2523.s: MT2523x startup file for GCC IDE.
    • GCC/syscalls.c: The minimal implementation of the system calls.
    • MDK-ARM/startup_mt2523.s: MT2523x startup file for Keil IDE.
  • Project configuration files using GCC
    • GCC/feature.mk: Feature configuration.
    • GCC/Makefile.: Makefile.
    • GCC/flash.ld: Linker script.
Run the demo
  • Connect the board to a PC with a USB cable.
  • Build this example project and wifi5932_ref_design project of mt2523_hdk, download the two binary files to the LinkIt 2523 HDK.
  • Reboot the 2523 HDK, it starts download the image to 5932 HDK.
  • When download complete, then 5932 HDK will boot up.
