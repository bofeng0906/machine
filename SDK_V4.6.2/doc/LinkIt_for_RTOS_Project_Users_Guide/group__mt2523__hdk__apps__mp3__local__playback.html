mp3_local_playback
mt2523_hdk » apps
Overview
  • Application description
    • This is a reference application to demonstrate usage of mp3_local_playback.
Hardware and software environment
  • Supported platform
    • LinkIt 2523 HDK.
  • Environment configuration
    • Need a serial tool (such as Tera terminal) for UART logging.
    • COM port settings: baudrate: 115200, data bits: 8, stop bit: 1, parity: none and flow control: off.
Directory contents
  • Source and header files
    • src/main.c: Main program.
    • src/at_command_serial_port.c: at command port server file.
    • src/bt_init.c: Bluetooth initialization file.
    • src/audio_player_main.c: MP3 player file.
    • src/sys_init.c: system init file.
    • src/ept_eint_var.c: EINT assignment file with variable names.
    • src/ept_gpio_var.c: GPIO pin assignment with variable names.
    • src/system_mt2523.c: MT2523x system clock configuration file.
    • GCC/startup_mt2523.s: MT2523x startup file for GCC.
    • GCC/syscalls.c: MT2523x syscalls for GCC.
    • inc/task_def.h: Define the task stack size, queue length, project name, and priority for the application to create tasks.
    • inc/hal_feature_config.h: MT2523x feature configuration file.
    • inc/memory_map.h: MT2523x memory layout symbol file.
    • inc/ept_eint_drv.h: EINT configuration header file.
    • inc/ept_gpio_drv.h: GPIO pin assignment and configuration file.
    • inc/ept_keypad_drv.h: Keypad configuration header file.
    • inc/main.h: Main program header file.
    • inc/hci_log.h: HCI log header file.
    • inc/audio_player.h: MP3 player extern API header file.
    • inc/audio_player_main.h: MP3 player main header file.
  • Project configuration files using GCC.
    • GCC/feature.mk: Feature configuration.
    • GCC/Makefile: Makefile.
    • GCC/flash.ld: Linker script.
  • Project configuration files using Keil IDE.
    • MDK-ARM/mp3_local_playback.uvprojx: uVision5 project file. Contains the project structure in XML format.
    • MDK-ARM/mp3_local_playback.uvoptx: uVision5 project options. Contains the settings for the debugger, trace configuration, breakpoints, currently open files, etc.
    • MDK-ARM/flash.sct: Linker script.
  • Project configuration files using IAR.
    • EWARM/mp3_local_playback.ewd: IAR project options. Contains the settings for the debugger.
    • EWARM/mp3_local_playback.ewp: IAR project file. Contains the project structure in XML format.
    • EWARM/mp3_local_playback.eww: IAR workspace file. Contains project information.
    • EWARM/flash.icf: Linker script.
Run the example
  • Build the sample project with the command "./build.sh mt2523_hdk mp3_local_playback" from the SDK root folder on GCC, or build the project mp3_local_playback.uvprojx from the MDK-ARM folder on Keil, or build the project mp3_local_playback.eww from the EWARM folder on IAR, then download the binary file to LinkIt 2523 HDK.
  • To run the application.
    • AT+EAUDPLY=1: Play CMD, send again will pause music.
    • AT+EAUDVOL=1: Volume up; AT+EAUDVOL=0 Volume down.
    • AT+EAUDSKIP=1: Change next track; AT+EAUDSKIP=0: Change previous track.
