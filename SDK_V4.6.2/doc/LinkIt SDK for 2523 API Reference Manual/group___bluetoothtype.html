Terms and Acronyms

Terms Details
ATT Attribute Protocol, to discover, read and write attributes on a peer device.
GAP Generic Access Profile defines the generic procedures related to the discovery of Bluetooth devices and link management aspects of connecting to Bluetooth devices. It also defines procedures related to use of different security levels.
GATT Generic Attribute Profile defines a service framework using the attribute Protocol for discovering services, and for reading and writing characteristic value on a remote device.
HCI Host Controller Interface. For more information, please refer to Wikipedia.
L2CAP Logical Link Control And Adaptation Protocol Specification. For more information, please refer to Wikipedia.
SM Security Manager. It defines the protocol and behavior to manage pairing, authentication and encryption between LE-only or BR/EDR/LE devices.
