enum hal_dvfs_freq_relation_t

This enum defines the relationship between the target frequency and the final frequency.

Enumerator
HAL_DVFS_FREQ_RELATION_L 

The lowest frequency at or above the target level.

HAL_DVFS_FREQ_RELATION_H 

The highest frequency below or at the target level.

enum hal_dvfs_status_t

This enum defines return status of certain DVFS HAL APIs.

User should check return value after calling these APIs.

Enumerator
HAL_DVFS_STATUS_NOT_PERMITTED 

The operation is not permitted.

HAL_DVFS_STATUS_BUSY 

Device or resource is busy.

HAL_DVFS_STATUS_UNINITIALIZED 

Non-initialized.

HAL_DVFS_STATUS_INVALID_PARAM 

Invalid parameter value.

HAL_DVFS_STATUS_ERROR 

The DVFS function detected a common error.

HAL_DVFS_STATUS_OK 

The DVFS function executed successfully.
