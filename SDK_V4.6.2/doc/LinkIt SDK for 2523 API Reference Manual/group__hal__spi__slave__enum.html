SPI slave transaction bit order definition.

Enumerator
HAL_SPI_SLAVE_LSB_FIRST 

Both send and receive data transfer is the LSB first.

HAL_SPI_SLAVE_MSB_FIRST 

Both send and receive data transfer is the MSB first.

enum hal_spi_slave_callback_event_t

This enum defines the SPI slave event when an interrupt occurs.

Enumerator
HAL_SPI_SLAVE_EVENT_POWER_ON 

Power on command is received.

HAL_SPI_SLAVE_EVENT_POWER_OFF 

Power off command is received.

HAL_SPI_SLAVE_EVENT_CRD_FINISH 

Configure read command is received.

HAL_SPI_SLAVE_EVENT_RD_FINISH 

Read command is received.

HAL_SPI_SLAVE_EVENT_CWR_FINISH 

Configure write command is received.

HAL_SPI_SLAVE_EVENT_WR_FINISH 

Write command is received.

HAL_SPI_SLAVE_EVENT_RD_ERR 

An error occurred during a read command.

HAL_SPI_SLAVE_EVENT_WR_ERR 

An error occurred during a write command.

HAL_SPI_SLAVE_EVENT_TIMEOUT_ERR 

A timeout is detected between configure read command and read command or configure write command and write command.

HAL_SPI_SLAVE_FSM_ERROR_PWROFF_AFTER_CW

HAL_SPI_SLAVE_FSM_ERROR_CW_AFTER_CR

HAL_SPI_SLAVE_FSM_ERROR_WRITE_AFTER_CR 

SPI slave error, received Write command after Configure Read command.

HAL_SPI_SLAVE_FSM_ERROR_READ_AFTER_CW
