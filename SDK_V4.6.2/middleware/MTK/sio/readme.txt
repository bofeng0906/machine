sio module usage guide

Brief:           This module is modem-sio interface, via ATCMD communication with modem.
Usage:           GCC: Include the module with "include $(SOURCE_DIR)/middleware/MTK/sio/module.mk"
                      in your GCC project Makefile.
Dependency:      Please define MTK_MODEM_ENABLE = y in the feature.mk under the project GCC folder.
Notice:          None.
Relative doc:    None.
Example project: Please find the project under project folder iot_modem_wifi5931_ref_design.
