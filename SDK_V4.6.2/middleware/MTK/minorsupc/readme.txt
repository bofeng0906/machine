minisupp module usage guide

Brief:          This module is the Wi-Fi supplicant implementation.
Usage:          GCC: Include the module with "include $(SOURCE_DIR)/middleware/MTK/minorsupc/module.mk" in your GCC project Makefile.
                     MTK_WIFI_CHIP_USE_MT5931 should set to "y".
Dependency:     None
Notice:         None
Relative doc:   Please refer to the website "http://w1.fi" for more detail.
Example project:Please find the project under project/mt2325_hdk/apps/iot_wifi5931
