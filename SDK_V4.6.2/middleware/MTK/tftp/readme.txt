TFTP module usage guide

Brief:          This module is an implementation of TFTP client.
Usage:          GCC: Include the module with "include $(SOURCE_DIR)/middleware/mtk/tftp/module.mk" in your GCC project Makefile.
                KEIL: Drag the middleware/MTK/tftp folder to your project. Add middleware/MTK/tftp/inc to include paths.
                IAR: Drag the middleware/MTK/tftp folder to your project. Add middleware/MTK/tftp/inc to include paths.
Dependency:     Please also include LWIP since TFTP uses it.
